# SubtranslatorReact

Gemini will translate your subtitle file.

A small React app to translate subtitle files (e.g. .srt, .vtt, .ass) using a Gemini-powered translation backend. The app preserves timing and formatting while translating subtitle text.

## Features
- Upload SRT/VTT/ASS subtitle files
- Automatic language detection
- Translate subtitle text while preserving timestamps
- Download translated subtitle file in the same format
